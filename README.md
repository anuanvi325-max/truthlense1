# Fake News Detection Project

## Project Structure
- data/fake_news_200.csv  --> dataset
- src/train_model.py      --> train model and save it
- src/predict_news.py     --> predict new news articles
- models/                 --> saved model and vectorizer

## Instructions
1. Open project in PyCharm.
2. Install required packages:
   pip install -r requirements.txt
3. Run training script:
   python src/train_model.py
4. Predict new news:
   python src/predict_news.py
